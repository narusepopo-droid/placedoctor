# 알림톡 템플릿

## 1. 알림 신청 완료

**템플릿 코드**: (승인 후 기입)

**강조 타이틀**: 순위 알림 신청 완료

**메시지 본문**:
```
[플레이스랭킹] 순위 알림 신청 완료

#{매장명}님, 순위 모니터링을 시작했어요.
이제 매주 #{요일}에 플레이스 키워드 순위 변화를 정리해 보내드릴게요.

#{추가문구}
```

**버튼**: 웹링크 - "순위 확인하기" → https://placeranking.com

---

## 2. 주간 순위 리포트

**템플릿 코드**: (승인 후 기입)

**강조 타이틀**: 이번주 플레이스 순위 리포트

**메시지 본문**:
```
[플레이스랭킹] #{매장명} 이번주 순위 리포트

대표 키워드 '#{키워드}'
#{지난순위}위 → #{이번순위}위

#{매장명}의 플레이스 전체 키워드별 
순위변화를 확인하실 수 있습니다.

경쟁 매장 변화까지 리포트로 
정리해 두었으니 확인해 보세요.

#{추가문구}
```

**버튼**: 웹링크 - "전체 리포트 보기" → https://placeranking.com

---

## 변수 설명

| 변수 | 설명 | 예시 |
|------|------|------|
| #{매장명} | 매장 이름 | 숲속카페 |
| #{요일} | 발송 요일 | 월요일 |
| #{키워드} | 대표 키워드 | 동변동 대형카페 |
| #{지난순위} | 지난주 순위 | 3 |
| #{이번순위} | 이번주 순위 | 1 |
| #{추가문구} | 관리자 페이지에서 자유 수정 | (빈 값 가능) |
